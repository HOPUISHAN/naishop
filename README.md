# naishop

* 引用了别人一个开源项目来学习，仿了一个美甲商城  
* 原生态的微信小程序，未使用框架

## 功能列表

* 首页  
* 分类首页、分类商品、人气推荐商品页面  
* 详情页面，预订页面  
* 搜索功能  
* 登录功能  
* 获取当前位置功能  
* 下滑加载更多
